--cache-rr=<rrtype>[,<rrtype>...]
By default, dnsmasq caches A, AAAA, CNAME and SRV DNS record types. This option adds other record types to the cache. The RR-type can be given as a name such as TXT or MX or a decimal number. A single --cache-rr option can take a comma-separated list of RR-types and more than one --cache-rr option is allowed. Use --cache-rr=ANY to enable caching for all RR-types.
